Welcome to the 24 Game

The objective of the game is to manipulate four given numbers using basic arithmetic operations (addition, subtraction, multiplication, division) to reach the number 24.

Rules:

  • You must use each of the four numbers exactly once.
  • You can use the operations in any order.
  • If you successfully reach 24, a new set of numbers will be presented.
  • Your score will increase each time you successfully reach 24.
